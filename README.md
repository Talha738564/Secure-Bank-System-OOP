# Secure Bank System (OOP Practice Project)

A small banking system built to practice and apply core Object-Oriented Programming concepts in Python.

## Concepts Covered
- [ ] Encapsulation (`@property`, private attributes)
- [ ] Inheritance (Account → SavingsAccount, CheckingAccount)
- [ ] Multiple Inheritance & MRO (RewardsMixin)
- [ ] Composition (Bank has-a list of Accounts)
- [ ] Polymorphism (apply_monthly_interest across account types)
- [ ] Dunder methods (`__str__`, `__eq__`, `__lt__`)
- [ ] Class vs Instance variables
- [ ] Static & Class methods
